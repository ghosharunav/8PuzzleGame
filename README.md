A classic 8 Puzzle Game built for Android using Kotlin and Java in Android Studio.
The game challenges players to arrange numbered tiles in the correct order by sliding them into the empty space.

The app includes two game modes, a timer, move counter, and an AI-powered optimal moves calculator using the A* search algorithm.

📱 Features
🎮 Two Game Modes

Classic Mode – Arrange tiles from 1 → 8

Reverse Mode – Arrange tiles from 8 → 1

⏱ Timer

Tracks the total time taken to solve the puzzle.

🔢 Move Counter

Counts every move made by the player.

🤖 Optimal Moves Calculator

Uses the A* Search Algorithm with Manhattan Distance heuristic to calculate the minimum number of moves required to solve the puzzle from the current board state.

🎯 Smart Game Controls

START button – Begin the timer when ready

NEW GAME button – Generate a new solvable puzzle

MODE selection – Switch between puzzle modes

🎉 Win Dialog

Displays final time and move count after solving the puzzle.

✨ Splash Screen

Animated splash screen displayed when the app launches.

🎨 Dual Theme System

Blue theme for Classic Mode

Orange theme for Reverse Mode
